# Frontend (Vite + React + TypeScript)

Rodar em desenvolvimento:

```bash
cd frontend
npm install
npm run dev
```

A configuração de desenvolvimento proxya `/api` para `http://localhost:4000`.

O frontend tem uma interface simples para gerenciar notas via API (listar, criar, editar, excluir).

Rodar:

```bash
cd frontend
npm install
npm run dev

# abrir http://localhost:5173
```

Os endpoints usados são os do backend descritos em `backend/README.md`.
